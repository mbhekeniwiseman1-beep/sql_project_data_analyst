# 📊 Data Analyst Job Market Analysis

A SQL project analysing the data analyst job market to uncover insights about salaries, in-demand skills, and the most strategic skills to pursue for a career in data analytics. Built as a portfolio piece to demonstrate real-world SQL problem-solving and analytical thinking on a relational database.

---

## 🛠️ Tools Used

- **Microsoft SQL Server** — all queries written and executed in SQL Server
- **SQL Server Management Studio (SSMS)** — query development and testing
- **GitHub** — version control and project sharing

---

## 🔍 Questions Explored

### 1. What are the top-paying remote Data Analyst jobs?
Identifies the top 10 highest-paying remote Data Analyst roles with specified salaries, giving a clear picture of the earning ceiling in the field.

### 2. What skills are required for the top-paying roles?
Joins skill data onto those top 10 jobs to reveal which specific tools and technologies are tied to the highest salaries.

### 3. What are the most in-demand skills for Data Analysts?
Scans all remote Data Analyst postings to find the top 5 most frequently requested skills — a direct signal of what employers value most.

### 4. What skills are associated with the highest average salaries?
Ranks the top 25 skills by average salary for remote Data Analyst roles, highlighting which skills are most financially rewarding to develop.

### 5. What are the most optimal skills to learn?
Combines demand and salary data to surface skills that offer both high job security and strong compensation — the strategic sweet spot for career growth.

## 💡 Key Takeaways

- High-paying remote Data Analyst roles tend to require specialized tools beyond standard SQL and Excel
- Niche and cloud-based skills can command significantly higher average salaries even when demand is lower
- The most optimal skills to learn balance both high demand and competitive pay — not just one or the other

## 🚀 How to Run

1. Load the job postings dataset into your Microsoft SQL Server instance
2. Open `data_analyst_jobs_analysis.sql` in SSMS
3. Run each query block individually to explore the results
